## 📖 项目介绍

高考填志愿、大学转专业、考研选方向……**选专业这件事，真的很难。**

「专业星球」是一个**娱乐向的专业倾向测试**，借鉴 MBTI 的 4 维度设计思路，通过 24 道生活情境题，帮你发现自己的：

- 🧠 思维方式：逻辑分析 vs 创意发散
- 👥 工作方式：独立钻研 vs 团队协作
- 🔬 兴趣导向：理论研究 vs 实践应用
- ⚖️ 价值追求：稳定安全 vs 挑战冒险

最终为你生成专属的 **4 字母人格码**（如 `LIPA`）、稀有度标签、个人档案，以及 **3 个量身定制的专业推荐**。

> ⚠️ **本测试仅供娱乐参考**，不能替代专业的职业规划和学业咨询。

---

## ✨ 项目特色

- 🎯 **24 道情境化题目**：不是枯燥的打分题，而是真实生活场景选择
- 🧬 **4 维度 × 3 选项**：比二元选择更细腻，结果更精准
- 💎 **稀有度系统**：「全球仅 3% 的人和你一样」，增强社交分享欲
- 📊 **详细个人档案**：关键词、精神动物、代表颜色、能量来源、同类名人
- 🎓 **专业深度推荐**：含适配度、就业方向、学习难度、避坑提醒
- 📤 **一键分享**：支持 Web Share API 和剪贴板复制
- 📱 **完全响应式**：手机、平板、桌面端都丝滑
- ⚡ **零依赖**：纯 HTML/CSS/JS，无需构建，秒开
